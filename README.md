# Homebrew Tap for storage.to

Official [Homebrew](https://brew.sh) tap for [storage.to](https://storage.to) CLI tools.

## Installation

```bash
brew install storageto/tap/storageto
```

Or:

```bash
brew tap storageto/tap
brew install storageto
```

## Usage

```bash
# Upload a file
storageto upload photo.jpg

# Upload multiple files as a collection
storageto upload *.log

# Show version
storageto version
```

## Updating

```bash
brew upgrade storageto
```

## Uninstalling

```bash
brew uninstall storageto
brew untap storageto/tap  # optional
```
